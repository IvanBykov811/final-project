## Опис
Створити web-аплікацію, яка дозволятиме формувати списки справ, які треба виконати протягом дня. Необхідно передбачити:
- можливість створення різної кількості списків справ;
- можливість додавати та видаляти справи;
- відмічати справи як виконані;
- збереження даних у **LocalStorage**, щоб списки не зникали після перезавантаження сторінки.

## Функціонал
- Додавання нових справ.
- Перемикання статусу “виконано/не виконано”.
- Збереження стану списку в LocalStorage.


## Технології
- **HTML5**, **CSS3**, **JavaScript**
- Збереження даних: **LocalStorage** браузера
